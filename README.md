# Homework Collection - Remastered

Due to unfamiliar with golang, I ran into mess of code. In this remaster, I want to reorganize and
write clean code in cpp.

## Quickstart

```bash
cmake -Bbuild -S. -GNinja
cmake --build build
cd build
./hc
```

## Prerequisite

You should install `cmake`.
